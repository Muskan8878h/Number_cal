# Number Calculator

A web-based calculator that allows users to:

- Enter numbers or expressions (e.g., 200+300)
- Automatically calculate values
- Calculate column totals
- Calculate Total D
- Calculate Total A
- Calculate Total A + D
- Highlight values greater than 1000
- Save multiple tables
- Load saved tables
- Edit saved tables
- Delete saved tables

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage
